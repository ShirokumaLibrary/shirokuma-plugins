# GitHub 出力言語

## ルール

GitHub に書き込む全てのコンテンツ（Issues, PR, Discussions のコメント・本文）は**日本語**で記述する。

| 要素 | 言語 |
|------|------|
| Issue / PR / Discussion の本文・コメント | 日本語 |
| PR タイトル（プレフィックス後） | 日本語 |
| コミットメッセージ（プレフィックス後） | 日本語 |
| Conventional commit プレフィックス (`feat:`, `fix:` 等) | English |
| コード / 変数名 | English |
| ラベル名 | English |
