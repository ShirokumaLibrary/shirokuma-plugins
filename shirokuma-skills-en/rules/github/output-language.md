# GitHub Output Language

## Rule

All content written to GitHub (Issues, PRs, Discussions comments and bodies) MUST be in **English**.

| Element | Language |
|---------|----------|
| Issue / PR / Discussion body and comments | English |
| PR title (after prefix) | English |
| Commit message (after prefix) | English |
| Conventional commit prefixes (`feat:`, `fix:`, etc.) | English |
| Code / Variable names | English |
| Label names | English |
