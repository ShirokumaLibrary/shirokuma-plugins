# reviewing-on-pr 設計メモ

スキル管理用メタデータ。実行時には読み込まれない。

## コンセプト

PR レビューのオーケストレーター。`review-worker` によるコードレビュー実行と、未解決レビュースレッドの分類・修正・返信・解決を自動チェーンで処理する。

## ユーザー制御ポイント (UCP)

`review-worker` 完了後、修正方針はユーザーが事前確認すべき意思決定のため、ステップ 2b でユーザー確認を挟む。

## 関連スキル

| スキル | 関係 |
|--------|------|
| `review-worker` | コードレビュー実行の委任先 |
| `coding-worker` | コード修正の委任先 |
| `commit-worker` | コミット・プッシュの委任先 |
