# review-worker 設計メモ

エージェント管理用メタデータ。実行時には読み込まれない。

## コンセプト

専門ロール別の包括的レビューを実行するサブエージェント。オーケストレーターとして複数ロール（code, security, testing, docs, plan 等）を順次実行し、全結果を統合して最終判断コメントを投稿する。

## 設計判断

- モデルは Opus（レビュー品質に高度な推論が必要）
- Edit ツールあり（レビュー指摘の自動修正提案用）
- Bash あり（テスト実行、lint 実行等の検証用）
- Web アクセスあり（セキュリティ脆弱性の照合用）
