# project-config-generator 設計メモ

スキル管理用メタデータ。実行時には読み込まれない。

## コンセプト

テックスタック検出に基づくプロジェクト固有設定の自動生成。`package.json` やプロジェクトファイルからスタックを検出し、スキルの `project/` ディレクトリに設定ファイルを展開する。

### 生成対象

| ファイル | 必須 | 内容 |
|---------|------|------|
| `tech-stack.md` | はい | フレームワーク・ライブラリのバージョン |
| `known-issues.md` | はい | 既知の問題・CVE |
| `patterns/` | いいえ | プロジェクト固有パターン |
| `setup/` | いいえ | セットアップ手順 |
| `workflows/` | いいえ | ワークフロー設定 |

### テンプレート変数置換

`{{NEXTJS_VERSION}}`, `{{REACT_VERSION}}` 等を `package.json` から自動解決。

## トリガーキーワード

SKILL.md の frontmatter `description` フィールドに定義。スキル選択はこのフィールドでマッチングされる。

- プロジェクトセットアップ、プロジェクト規約の更新時に使用
- `setting-up-project` から委任されることが多い
